# Brienz/Brinzauls Landslide Monitor

A real-time monitoring dashboard for the Brienz/Brinzauls landslide area. This web application provides a centralized view of critical data, including live camera feeds, geographical information, and sensor data visualizations.

## Features

- **Live Monitoring**: Integration with live camera feeds to visually monitor the area.
- **Interactive Map**: detailed map view powered by [Leaflet.js](https://leafletjs.com/) for geographical context.
- **Data Visualization**: Charts and graphs using [Chart.js](https://www.chartjs.org/) to display sensor data (e.g., weather, movement).
- **Modern UI**: A responsive, dark-themed interface built with [Tailwind CSS](https://tailwindcss.com/).
- **Real-time Info**: Live clock and status indicators.

## Technologies Used

- **HTML5 & JavaScript**: Core structure and logic.
- **Tailwind CSS**: For styling and responsive design (via CDN).
- **Leaflet.js**: For interactive maps (via CDN).
- **Chart.js**: For data visualization (via CDN).

## Getting Started

Since this is a static web page, no build process is required.

1. Clone the repository:
   ```bash
   git clone https://github.com/RicPedre/Brienz-Monitor.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Brienz-Monitor
   ```
3. Open `index.html` in your preferred web browser.
